# comfyui-site-packages-clean-cache
comfyui-site-packages-clean-cache

下载comfyui-site-packages-clean-cache.bat

例如 ：D:\ComfyUI-PyTorch-2.8.0+cu128 

复制 D:\ComfyUI-PyTorch-2.8.0+cu128\comfyui-site-packages-clean-cache.bat
自动清理 D:\ComfyUI-PyTorch-2.8.0+cu128\python\Lib\site-packages 
下面的多余的 以前老版本的 依赖包 

主要是删除 pip 升级残留文件夹（如 ~~mpy, ~umpy, ~orch 等）
